# Tetris w Pythonie

## Opis Aplikacji

Program jest wzorcem popularnej gry Tetris napisanej w języku **Python** i wykonany za pomocą zewnętrznej biblioteki do tworzenia gier **Pygame**

## Instrukcja instalacji i uruchomienia

Proszę pobrać całe repozytorium. Następnie w repozytorium znaleźć plik o nazwie 
```run.bat``` i go kliknąć.
Po kliknięciu program się załaduje i uruchomi.

## Wykorzystane technologie
* Język programowania - *Python*
* Biblioteka do tworzenia gier w języku Python - *Pygame*
* Plik uruchamiający program - *.BAT*
